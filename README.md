# US Stock Market Analysis with Python & Pandas

Exploratory analysis of 5 US stocks (AAPL, MSFT, GOOGL, JPM, BAC) 
covering the 2023–2025 period.

Key analyses:
- Daily and cumulative returns
- 20-day and 60-day moving averages
- Correlation matrix for portfolio diversification
- Annualized risk, return, and Sharpe Ratio per asset

Tools: Python, Pandas, yfinance, Matplotlib, Seaborn
